# Skill: Thiết kế giao diện web thuần cho Antigravity

## Cách dùng

1. Copy thư mục `.agent/skills/thiet-ke-giao-dien-web-thuan` vào workspace Antigravity của bạn.
2. Mở workspace bằng Antigravity.
3. Ra lệnh bằng tiếng Việt, ví dụ:

```text
Tạo cho tôi bộ giao diện landing page cho Brand BKAP AI, màu chủ đạo xanh dương, phong cách Flat hiện đại. Project tên bkap-ai-landing. Cần các trang index, about, courses, contact. Dùng HTML, CSS, JS, TailwindCSS, jQuery và toastr.js. Tạo trong projects/bkap-ai-landing.
```

## Skill sẽ làm gì?

- Phân tích yêu cầu.
- Lập kế hoạch thiết kế.
- Tạo thư mục `projects/<ten_project>`.
- Tạo các file HTML/CSS/JS/images/assets.
- Dùng TailwindCSS, jQuery, toastr.js hoặc plugin phù hợp.
- Đảm bảo giao diện responsive và đúng yêu cầu.
